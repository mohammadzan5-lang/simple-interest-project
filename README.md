# Simple Interest Calculator

## Description
This project is a simple interest calculator that calculates the interest based on the principal amount, rate of interest, and time period.

## How it works
The calculator takes three inputs:
- Principal amount (P)
- Rate of interest (R)
- Time period (T)

It then calculates the simple interest using the formula:

Simple Interest = (P × R × T) / 100

## Usage
Run the script and enter the required values when prompted.

## Example
If:
- Principal = 1000
- Rate = 5%
- Time = 2 years

Then:
Simple Interest = (1000 × 5 × 2) / 100 = 100
